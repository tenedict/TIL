# 자바 스크립트 기본

    - 변수선언  let, const (const는 재할당이 불가능, var는 쓰지마셈)
    - 할당 =
    - 할당 + 선언 let 아무단어 = 값
    - 자바 스크립트는 모든 선언을 호이스팅함
        - 호이스팅은 위로 올리는 것 

    - 문자열
        - 작따 큰따 다가능
        - 템플릿 리터럴
            - 따옴표대신 백틱씀
            - ${}형태로 씀

    - 연산자
        - ++ +=1 이라는뜻 --도 있음

    - 동등 비교할때
        - ==
            - 타입을 일치시켜버리고 비교
        - ===
            - 엄격한 비교 이거 주로 사용

    - 논리 비교
        - and
            - &&
        - or
            - ||
        - not
            - !

    - 삼항 연산자
